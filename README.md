## Installation

- Node.js (v20.14.01 or later)
- npm or yarn

### Steps


1. **Clone the repository**:
    git clone https://github.com/Tusharagarwal321/ProductLandingPage.git
    cd ProductLandingPage


2. **Install dependencies**:
    npm install
    # or
    yarn install
  

3. **Start the development server**:
    npm run dev
    # or
    yarn dev
